The TX16S simulator fixtures are a curated subset of a real SD-card backup.

Included:
- Simulator-oriented model YAML files and their model images.
- The Sunbeam theme from the backup, available on the fixture SD card.

Excluded:
- Real aircraft, HIL/test-only models, firmware binaries, logs, screenshots, OS metadata, compiled Lua, and unrelated scripts/widgets.
- Hardware-specific calibration values from the backup; simulator fixtures keep neutral calibration spans.
